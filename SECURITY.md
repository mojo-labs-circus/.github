# Security Policy

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Email <clarkehines@icloud.com> with the subject line `[SECURITY] mojo-labs-circus/<repo>`. Include:

- Which repo and component is affected
- A description of the vulnerability
- Steps to reproduce
- Potential impact

You'll receive a response within 7 days. If the issue is confirmed, a fix will be prioritised and you'll be credited in the release notes unless you prefer otherwise.
